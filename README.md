# Gastronom AI Documentation

Documentation for the conversational dataset pipeline used to process customer conversations from WhatsApp, Instagram and Email.

The system extracts conversations, classifies them using AI and stores structured records in Airtable for analytics and training.

# Gastronom AI Documentation
