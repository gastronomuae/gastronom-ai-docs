# Dataset Schema

Each conversation is stored as a single record in Airtable.

## Core Fields

conversation_id  
Unique identifier of the conversation.

channel  
WhatsApp / Instagram / Email

customer_phone  
Customer phone number when available.

message_text  
Conversation content.

confidence_score  
AI classification confidence score.

issue_category  
Category assigned by AI.

resolution_status  
resolved / unresolved / pending

conversation_status  
open / closed
